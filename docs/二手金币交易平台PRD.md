# 二手金币交易平台产品需求文档 (PRD)

## 一、产品定位和目标用户

### 1.1 产品定位

**产品名称**：金币汇 - 二手金币交易平台

**产品定位**：专业的二手实物金币/纪念币C2C交易服务平台，为金币收藏爱好者、投资者提供安全、透明、便捷的交易渠道。

**核心价值主张**：
- 真实性保障：专业鉴定机制，杜绝假币流通
- 价格透明：市场行情实时展示，交易价格公开透明
- 交易安全：资金托管、担保交易，降低交易风险
- 便捷流通：解决二手金币交易信息不对称、渠道少的问题

### 1.2 目标用户画像

#### 核心用户群体

| 用户类型 | 用户画像 | 核心需求 | 使用场景 |
|---------|---------|---------|---------|
| **收藏爱好者** | 30-55岁，中高收入，有收藏爱好 | 寻找稀缺币种、品相好的金币，专业交流 | 收藏投资、主题收藏、礼品馈赠 |
| **投资者** | 25-50岁，有一定投资经验，追求资产多元化 | 了解市场行情，低价买入高价卖出 | 资产配置、避险投资、短期套利 |
| **变现卖家** | 25-60岁，持有金币需要变现 | 快速出手、合理定价、安全收款 | 礼品变现、资产周转、收藏升级 |
| **礼品买家** | 25-45岁，有礼品采购需求 | 品相好、包装精美、性价比高 | 节日礼品、商务馈赠、婚庆喜事 |

#### 用户规模估算

- 种子用户（首年）：5,000 - 10,000 人
- 目标用户（3年）：50,000 - 100,000 人
- 潜在市场规模：中国金币收藏市场约500亿元/年

## 三、用户故事

### 3.1 P0 优先级用户故事

#### 故事 1：注册登录

**作为** 新用户
**我希望** 能通过手机号或微信快速注册登录
**以便** 开始使用平台进行金币交易

**验收标准**：
- [ ] 支持手机号+验证码注册登录
- [ ] 支持微信一键授权登录
- [ ] 登录后生成用户唯一ID
- [ ] 新用户完成注册获得新手引导

---

#### 故事 2：实名认证

**作为** 需要进行交易的用户
**我希望** 完成实名认证
**以便** 获得更高的交易额度和信任度

**验收标准**：
- [ ] 支持身份证正反面上传
- [ ] 支持人脸识别验证
- [ ] 审核时间不超过24小时
- [ ] 认证通过后显示认证标识

---

#### 故事 3：发布商品

**作为** 卖家
**我希望** 能方便地发布我的金币商品
**以便** 快速找到买家

**验收标准**：
- [ ] 支持上传至少5张商品图片
- [ ] 填写金币基本信息：名称、年份、面值、品相、重量
- [ ] 设置商品价格（一口价）
- [ ] 填写商品描述
- [ ] 选择是否提供鉴定证书
- [ ] 发布成功后进入待审核状态

---

#### 故事 4：浏览搜索商品

**作为** 买家
**我希望** 能快速找到我想要的金币
**以便** 进行购买决策

**验收标准**：
- [ ] 支持按分类浏览（年份、面值、题材、品相）
- [ ] 支持关键词搜索
- [ ] 支持价格区间筛选
- [ ] 支持按价格、时间、热度排序
- [ ] 显示商品缩略图、价格、品相信息

---

#### 故事 5：下单购买

**作为** 买家
**我希望** 能顺利下单购买商品
**以便** 完成交易获得金币

**验收标准**：
- [ ] 选择收货地址
- [ ] 确认订单信息
- [ ] 选择支付方式
- [ ] 支付成功后订单状态变更为"已支付"
- [ ] 卖家收到新订单通知

---

#### 故事 6：订单管理

**作为** 买卖双方
**我希望** 能查看和管理我的订单
**以便** 了解交易进度

**验收标准**：
- [ ] 订单列表按状态分类显示
- [ ] 订单状态：待付款、待发货、待收货、已完成、已取消、退款中
- [ ] 查看订单详情：商品信息、物流信息、交易信息
- [ ] 买家可确认收货
- [ ] 卖家可发货、查看收入

---

#### 故事 7：资金托管

**作为** 买家
**我希望** 我的付款由平台托管
**以便** 确认收货后才打款给卖家，保障资金安全

**验收标准**：
- [ ] 买家付款后资金进入托管账户
- [ ] 买家确认收货后资金打给卖家
- [ ] 退款时资金原路返回
- [ ] 显示资金托管状态

---

**以便** 帮助其他用户决策，促进优质服务

#### 故事 10：用户等级体系

**作为** 活跃用户
**我希望** 我的交易行为能提升用户等级
**以便** 获得更多平台权益和信任度

**验收标准**：

- [ ] 根据交易次数、交易金额、信用评分升级

## 八、总结

本产品需求文档定义了"金币汇"二手金币交易平台的核心功能、用户故事和优先级。产品定位于解决二手金币交易中的**信任、安全、便捷**三大痛点，通过**实名认证、资金托管、鉴定服务**建立交易信任体系。

**核心差异化优势**：
1. 专业鉴定服务，解决假币问题
2. 资金托管保障，降低交易风险
3. 信用评价体系，建立买卖信任
