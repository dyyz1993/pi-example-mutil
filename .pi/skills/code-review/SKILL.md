# Code Review Skill

## 使用场景
- 审查新写的代码
- 检查 PR 变更
- 发现潜在问题

## 审查清单

### 代码质量
- [ ] 代码是否清晰易读
- [ ] 命名是否有意义
- [ ] 是否有重复代码
- [ ] 函数是否过长

### 安全性
- [ ] 是否有 SQL 注入风险
- [ ] 是否有 XSS 风险
- [ ] 敏感数据是否加密
- [ ] 输入是否验证

### 性能
- [ ] 是否有 N+1 查询
- [ ] 是否有不必要的循环
- [ ] 是否有内存泄漏风险
